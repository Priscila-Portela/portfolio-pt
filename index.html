Projetos de Ciências de Dados

Previsão de Vendas da Rossmann Utilizando Machine Learning e Implementação via Bot no Telegram

Este projeto de ciência de dados envolve a previsão das vendas das próximas seis semanas da loja Rossmann. O processo incluiu a realização de feature engineering, análise exploratória de dados (EDA), preparação de dados e seleção de características através do método Boruta. Foram treinados algoritmos de regressão linear, Random Forest Regressor e XGBoost, com finetuning realizado utilizando o método de Random Search.

O modelo preditivo foi então validado e implementado em produção através de um bot no Telegram, facilitando o acesso às previsões em tempo real através de qualquer celular com acesso à internet.

As ferramentas utilizadas foram:
  • Python 3.8, Scikit-learn e Flask
  • Métricas de performance: Accuracy, Precision, Recall e F1-Score
  • Algoritmos de regressão: Linear Regression, Regressor, Random Forest Regressor, Polinomial Regression, Linear Regression Lasso, Random Forest Regressor, XGBoost
  • Métricas de performance: RMSE, MAE e MAPE

Ensaio de Machine Learning

Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento, através de um ensaio de Machine Learning.

O resultado final do projeto foi um painel mostrando a performance desses algoritmos de Machine Learning, a partir da variações dos valores dos principais parâmetros de cada algoritmo, que controlam o estado de overfitting e underfitting.

As ferramentas utilizadas foram:
  • Python 3.8 e Scikit-learn
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression
  • Métricas de performance: Accuracy, Precision, Recall e F1-Score
  • Algoritmos de regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polinomial Regression, Linear Regression Lasso, Linear Regression Ridge, Linear Regression Elastic Net, Polinomial Regression Lasso, Polinomial Regression Ridge e Polinomial Regression Elastic Ne
  • Métricas de performance: R2, MSE, RMSE, MAE e MAPE
  • Algoritmos de agrupamento: K-Means e Affinity Propagation
  • Métricas de performance: Silhouette Score
